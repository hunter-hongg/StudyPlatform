Tmux使用方法
============

安装
------------
sudo apt install tmux 

新建会话
------------
1. tmux new -s <session_name>

删除会话
------------
1. tmux kill-session删除会话

会话内新建窗口
------------
1. <C-b><C>新建

脱离会话
------------
1. <C-b><D>脱离

连接会话
------------
1. tmux a连接

切换窗口
------------
1. <C-b><W>切换
2. <C-b><NUM>在当前会话内切换，NUM可以是0-9

列出会话
------------
1. tmux ls列出
