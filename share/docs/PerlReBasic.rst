Perl正则表达式基本
==================

元字符.
------------
1. 元字符.表示任意一个字符
2. .不能匹配换行符

元字符\
------------
1. 元字符\表示转义
2. 例如，\.表示真正的.
3. \\表示真正的\

元字符?
------------
1. 元字符?表示前一个字符可有可无
2. 例如，/ac?b/能匹配ab和acb
3. 默认只对前一个字符生效

元字符+
------------
1. 元字符+表示前一个字符出现任意次
2. 但前一个字符至少出现一次
3. 例如，/ac+b/能匹配acb和accb，但无法匹配ab 
4. 同?，默认对前一个字符生效

元字符*
------------
1. 元字符*表前一个字符出现任意次
2. 与+不同，允许出现0次
3. 例如，/ac*b/能匹配acb和accb，同时也能匹配ab 
4. 同+和?，默认对前一个字符生效
