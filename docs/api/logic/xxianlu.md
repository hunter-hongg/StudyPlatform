# API文档-logic/xxianlu.dart

## `class XXianLu`
* 仙禄**统一处理类**
* 所有方法均为**静态**方法

## `static int getXianLu(int level)`
* **功能**: 根据当前等级计算仙禄。
* `level`: 当前等级
* **返回值**:
  * `int`: 计算得出的仙禄

## `static int getLevelNeed(int levelNow)`
* **功能**: 根据当前等级计算升级需要的仙币。
* `levelNow`: 当前等级
* **返回值**:
  * `int`: 计算得出的升级需要的仙币
