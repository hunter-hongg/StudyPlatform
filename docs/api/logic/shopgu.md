# API文档-logic/shopgu.dart

## `class Shopgu`
- **古玩商城**统一处理类

## `static void dealA(BuildContext context, int minus, AddFile fp)`
- **AddFile类文件**处理方法
- `minus`为减少的白银
- `fp`为增加物品的文件

## `static void dealP(BuildContext context, int minus, FilePassword fp)`
- **FilePassword类文件**处理方法
- `minus`为减少的白银
- `fp`为增加物品的文件
